# project

Project with .venv, requirements.txt, and src/main.py.

Bonjour voici mon projet qui permet d'utiliser une bibliotheque.
On peut creer des user, gerer leurs emprunts, leurs abonnements et les livres et leurs exemplaires.
Les données peuvent etre save et load en JSON.
Il y'a un historique de livre, des genres et le systeme de multi-exemplaires avec reservation et file d'attente.
Les emprunts incluent une date limite de retour une gestion des pénalités et des historiques coté utilisateur et coté livre.
